Öğrenci Takip Sistemi

Bu proje, Python kullanılarak geliştirilmiş basit bir öğrenci takip sistemi uygulamasıdır. 
Uygulama, öğrenciler ve öğretmenler arasında temel etkileşimi sağlamak amacıyla tasarlanmıştır.

Proje Amacı
Python ile uygulama geliştirme pratiği yapmak
Kullanıcı yönetimi ve temel sistem mantığını öğrenmek
Öğrenci–öğretmen etkileşimini modelleyen bir yapı kurmak

Özellikler
Öğrenci ve öğretmen kayıt sistemi
Öğrencilerin öğretmenlerden randevu alabilmesi
Öğretmenlerin öğrencilere çalışma planı oluşturabilmesi

Kullanılan Teknolojiler
Python
PyCharm
